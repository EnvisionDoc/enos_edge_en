Edge Installation Guide
==========================

.. rubric:: Prerequisites
---------------------------

-   You must have set up the VPN connection from the edge installation environment to the EnOS Cloud.

-   You must have prepared your server to intall the edge. For the list of recommended hardware models for EnOS Edge, see the Appendix.

-   Ensure that the operating system running on the hardware server is Linux CentOS7.4
