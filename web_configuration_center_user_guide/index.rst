.. rubric:: Logging on to Web Configuration Center

To log in to the Web configuration center:

1.  Select the URL of the public cloud environment for your region according to the following table.

2.  Use the authorized username to login.

*Table: Web Configuration Center Address*

.. list-table::
   :header-rows: 1

   * - Environment
     - Address
   * - Europe
     - https://eoseutoolset.envisioncn.com/eostoolset/addbox/index.html
   * - US
     - https://eosustoolset.envisioncn.com/eostoolset/addbox/index.html
   * - China AWS
     - http://10.21.18.226:8080/eostoolset/addbox/index.html
   * - China Azure
     - http://10.24.8.39:8080/eostoolset/addbox/index.html
