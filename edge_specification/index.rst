Edge Specifications
====================

As the data acquisition front-end module of EnOS IoT platform, the main
functionality of EnOS Edge is to enbale the platform to connect devices
of various types, ingest device telemetry, normalize and standardize the
collected data, and then upload the processed data to the cloud
platform.
